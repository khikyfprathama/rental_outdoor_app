# CommitHike 🏔️ — Aplikasi Manajemen Rental Outdoor

CommitHike adalah aplikasi mobile modern yang dirancang untuk membantu bisnis rental dan penjualan perlengkapan outdoor. Dibangun menggunakan Flutter dan Material 3, aplikasi ini menyediakan solusi sederhana, cepat, dan responsif untuk mengelola inventaris, transaksi penyewaan, serta audit stok barang.

Aplikasi mendukung mode terang (Light Mode) dan gelap (Dark Mode) dengan antarmuka yang modern dan nyaman digunakan.

---

## ✨ Fitur Utama

### 📊 Dashboard

* Tampilan ringkasan data bisnis secara real-time.
* Statistik total barang, penyewaan aktif, dan pendapatan.
* Akses cepat ke fitur utama melalui Quick Actions.

### 📦 Manajemen Inventaris

* Kelola data perlengkapan outdoor dengan mudah.
* Kategorisasi barang untuk kebutuhan rental maupun penjualan.
* Pencarian dan filter data yang cepat.
* Tambah, ubah, dan hapus kategori barang.

### 📋 Stock Opname

* Pencatatan stok fisik secara langsung.
* Perbandingan otomatis antara stok sistem dan stok aktual.
* Indikator selisih stok secara real-time.

### 💳 Manajemen Rental

* Pencatatan transaksi penyewaan.
* Pemilihan periode sewa dengan perhitungan durasi otomatis.
* Riwayat transaksi lengkap.
* Tampilan invoice digital untuk setiap transaksi.

### 🎨 Tampilan Modern

* Material 3 Design.
* Dukungan Light Mode & Dark Mode.
* Desain responsif untuk berbagai ukuran layar.

## 🚀 Instalasi

Clone repository:

```bash
git clone https://github.com/khikyfprathama/rental_outdoor_app.git
cd rental_outdoor_app
```

Install dependencies:

```bash
flutter pub get
```

Jalankan aplikasi:

```bash
flutter run
```

---

## 📌 Tujuan Proyek

CommitHike dibuat untuk membantu pelaku usaha rental outdoor dalam mengelola inventaris, transaksi penyewaan, dan stok barang secara lebih efisien melalui satu aplikasi mobile yang sederhana dan modern.
